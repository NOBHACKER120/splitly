1. Bill Splitter ‚Äî quick steps

This is the standard flow to fairly split an expense among a group.

Steps

  • Enter number of people & click Next.
  • Add each person's Name and Amount paid.
  • Click Calculate. Splitly computes per-person share and shows the transactions needed (who pays whom).

Example & rounding

2. Random Splitter ‚Äî explanation

Random Splitter has three modes: Completely Random, Weighted and Percentage-range. Use it for fun-splits, lucky draws, or surprise contributions.

How to use

  • Enter total amount, number of people ‚Üí Next ‚Üí add names.
  • Choose a mode (Random / Weighted / Percentage) ‚Üí Split Now.
Tip: Weighted mode is useful when some contributors should pay more (eg. drivers, event organisers).

3. Create Group ‚Äî how it connects to the Group Dashboard

Groups let you save a reusable list of people (roommates, travel buddies, family). Creating a group here saves it and opens it in the Group Dashboard.

Flow

  1. Open Create Group ‚Üí enter group name & member count.
  2. Fill member names ‚Üí Save Group.
  3. Save action stores the group in localStorage and redirects to group_dashboard page where the new group is available immediately.
