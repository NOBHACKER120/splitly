How Splitly Works — Fast, Fair & Friendly

Why Use Splitly?

Splitly is built for real groups — friends, travelers, roommates, or event hosts. You can:

  • Add multiple expenses easily
  • Select who paid
  • Automatically calculate how much each person owes

Choose between equal split, weighted split, or even a fun “Random Split” mode to keep things lively.

Key Features

  • Quick Expense Entry Add items, notes, and choose the payer in seconds — no sign-up needed to try it.
  • Flexible Split Modes Equal Split, Weighted Split, and Random Split make bill-sharing flexible and fun.
  • Share & Export Generate a summary or share a settlement link — pay easily via UPI, bank, or other means.
  • Privacy First We keep data collection minimal. Sensitive info is encrypted and secure.

Pro Tips

  • Agree Before Using Random Mode If you're using Random Split, make sure everyone’s OK with it — it’s meant for fun, not complicated accounting.
  • Add Context with Notes Use short descriptions like “Dinner – Day 2” to make later settlements easier to understand.
  • Settle Often Clearing small balances regularly saves you from long, complex debts later.
